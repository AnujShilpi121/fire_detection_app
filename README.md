The forest fire detection system is being developed by using python and its libraries,modules, deep learning, and django (for frontend and backend ). 
The development of the project is in progress, the work is going on development to make it complete even though anybody can run the partially
developed project by using the below following steps:

step 1: Open the terminal or powershell window and move to the directory name fire_detection_app in which the manage.py is located.
step 2: Write the command there,
                             python manage.py runserver
step 3: Copy the returned url(http url) to the browser and hit enter
